项目概述
基于知名移动游戏"Cookie Cats"的A/B测试数据，分析将游戏内付费门槛从第30关推迟至第40关对玩家短期和长期留存率的影响。

业务问题
假设：推迟付费门槛可能改善新手体验（提升短期留存），但可能减少付费动机（降低长期留存）
目标：通过统计检验验证门槛位置调整的整体效果，为游戏设计提供数据支持

关键指标
主要指标：7日留存率（衡量长期用户价值）
次要指标：1日留存率（衡量新用户体验）
样本量：90,000+ 玩家

分析方法
卡方检验（比例差异检验）
α = 0.05
Python, Pandas, Scipy, Matplotlib, Numpy

主要结果
1. 1日留存率：无显著差异（p = 0.074）
2. 7日留存率：显著下降 0.8%（p = 0.0015）
3. 结论：不建议推迟付费门槛

发现一：短期留存率无显著变化（p = 0.074）
可能原因：大部分短期玩家在达到30关前已流失，未能触及付费门槛
数据支持：XX%的玩家游戏轮数 < 30关
业务启示：门槛位置对新手玩家体验影响有限，应更关注前30关的游戏设计

发现二：长期留存率显著下降（p = 0.0015）  
可能原因：40关设置打破了长期玩家的游戏习惯和心理预期
行为分析：玩家在适应无障碍游戏后，对突然出现的付费门槛产生抵触
对比分析：30关设置在玩家尚未形成固定习惯时引入挑战，接受度更高


技术栈
- Python 3.8+
- Pandas, NumPy
- Scipy, Matplotlib
